# /escape — エラー連鎖脱出

同じエラーが3回以上解決できない時に実行します。

## 実行手順

1. **現状を記録する**：`docs/escape.md` に以下を追記する

```
## <日時>
- 発生フェーズ: T-XX <タスク名>
- エラー内容: <エラーメッセージの要約>
- 試した対策: <箇条書き>
- 試行回数: X回
```

2. **ユーザーに報告する**：

```
【/escape 発動】
T-XX で X回試みましたが解決できませんでした。
エラー: <内容>

対処法：
1. VSCode ターミナルで `/clear` を実行してセッションをリセット
2. 再度 `claude --permission-mode bypassPermissions` で起動
3. 起動後に次を実行：
   「/escape後の再開です。docs/escape.md を読んで、Opus 4.7 で問題を解決してください」
```

3. **自分でできる場合**：Opus 4.7 への切り替えは `/model claude-opus-4-7` で可能（Pro プランの場合トークン消費大きいため最終手段）

## 使わないケース

- エラー1〜2回：通常のデバッグで続行
- 警告（Warning）のみ：ビルドが通れば続行可
