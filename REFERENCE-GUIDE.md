# REFERENCE-GUIDE.md

## Reference Gate

大きな UI/API/データモデル/アーキテクチャ変更の前に、実在例を最低3件確認し、`docs/decisions.md` に記録する。

## 記録項目

- URL
- 採用する要素
- 避ける要素
- WorkSim への適用判断
- 将来変更可能性

## 優先ソース

- Next.js 公式 docs。
- Supabase 公式 docs。
- Playwright 公式 docs。
- OpenAI Codex CLI 公式 docs。
- 公的機関や企業公式ページ。

## 禁止

- 出典なしで「一般的」として UI/API/DB を作らない。
- 古い記事だけを根拠にしない。
- npm package 名や CLI 機能名を未確認で書かない。
