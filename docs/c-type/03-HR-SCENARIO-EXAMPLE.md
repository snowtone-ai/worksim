# HR Scenario Example

## 1. 基本情報

- 企業名: 架空商事株式会社
- 業界: 商社
- 職種: 海外市場開拓
- 想定部署: 事業開発部
- 対象学生: 文理不問、海外ビジネスに関心がある学生
- 想定所要時間: 18分
- 難易度: 標準

## 2. このシナリオで体験させたい仕事

- 仕事の概要: 市場情報、現地代理店の状況、採算見込みを見て、初回提案の優先順位を決める。
- 学生に理解してほしいこと: 海外営業は語学だけではなく、情報の不確実性と採算リスクを扱う仕事である。
- 実際の業務に近いポイント: 販売見込み、物流条件、現地パートナーの信頼度を同時に見る。
- あえて簡略化してよいポイント: 為替計算や契約条項の詳細。

## 3. 学生の役割

- 学生が演じる立場: 若手の海外市場開拓担当
- 与えられている責任: 初回訪問で深掘りすべき論点を整理する
- 判断しなければならないこと: 売上可能性、リスク、関係構築のどれを優先するか

## 4. タスク前画面

- タスクタイトル: 新市場への初回提案準備
- タスク概要: 現地代理店から届いた販売見込みと制約条件を読み、訪問前に確認すべき優先論点を決める。
- 業界名: 商社
- 職種名: 海外市場開拓
- START前に伝えるべきこと: 正解を探すのではなく、限られた情報から仕事の進め方を選ぶ。
- 背景画像に入れたい雰囲気: 会議室、輸出資料、ノートPC、地図

## 5. 光るオブジェクト画面

- 確認させたい資料: 市場メモ、代理店メール、粗利試算表
- 確認させたい状況: 提案期限が近く、情報が一部不足している
- 汎用TODOメモ: 依頼内容を確認、必要な情報を整理、次の対応を判断
- クリック後に表示する情報: 代理店メール本文と粗利試算の要点

## 6. 選択肢画面

- タスクタイトル: 訪問前に何を優先確認するか
- 問題文・概要: 代理店は売上見込みを強調しているが、物流条件と販売後サポートの情報が不足している。
- 選択肢A: まず売上見込みを信じ、提案資料の見栄えを整える
- 選択肢B: 粗利と物流条件を確認し、成立条件を明確にしてから提案する
- 選択肢C: 関係構築を優先し、詳細条件は訪問後に整理する

## 7. 各選択肢の意味

- Aを選ぶ学生の傾向: 推進力はあるが、採算や実行条件の確認が薄くなりやすい
- Bを選ぶ学生の傾向: 数字と実行可能性を重視して判断できる
- Cを選ぶ学生の傾向: 関係構築を重視するが、判断材料の不足を先送りしやすい
- 正解・不正解ではなく評価したい観点: 不確実な情報をどう扱うか

## 8. 結果画面

- フィードバックA: 前に進める力がある一方、条件確認を後回しにするとリスクが残る
- フィードバックB: 商社の市場開拓で重要な採算と実行条件を押さえられている
- フィードバックC: 関係構築の視点は有効だが、事前に確認すべき数字を残さないことが重要
- 学生に持ち帰ってほしい学び: 海外市場開拓は熱意だけでなく、情報の不足を整理して意思決定する仕事である

## 9. 企業側が見たい観点

- 判断力: 不確実性の中で優先順位を決められるか
- 顧客理解: 代理店と最終顧客の違いを理解できるか
- 論理性: 売上と粗利を分けて考えられるか
- 協調性: 関係構築を軽視しないか
- 倫理観: 誇張した販売見込みをそのまま使わないか
- 主体性: 不足情報を自分から確認できるか

## 10. NG表現

- 実在顧客名: 使用しない
- 機密情報: 実数ではなく架空の簡易数値を使う
- 差別的表現: 使用しない
- 過度に専門的な社内用語: 初出で説明する
- 学生が理解できない略語: 略語だけで書かない
