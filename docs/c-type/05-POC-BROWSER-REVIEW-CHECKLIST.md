## Selection Screen

- [ ] `/play` で業界カードが見やすい
- [ ] `/play/[industry]` で職種名、説明、タグ、所要時間が揃っている
- [ ] 全50職種への導線が維持されている
- [ ] PCとモバイルでカードの文字がはみ出していない

## Task Start Screen

- [ ] 業界名と職種名が分かる
- [ ] タスク概要が2〜4行で理解できる
- [ ] 背景画像なしfallbackが安っぽく見えない
- [ ] 暗めoverlayで文字が読める
- [ ] STARTボタンが明確で押しやすい

## Glowing Object Screen

- [ ] 光る対象がクリック対象だと分かる
- [ ] TODOメモが汎用文言になっている
- [ ] Webエンジニア固有のPR、GitHub、コードレビュー等が共通UIに残っていない
- [ ] どの職種でも違和感が少ない

## Choice Screen

- [ ] タスクタイトルが表示される
- [ ] 問題文・概要が表示される
- [ ] 3択が同じ重みのカードで表示される
- [ ] 選択後の状態が分かる
- [ ] 次へ進むCTAが分かる

## Result Screen

- [ ] 結果の納得感がある
- [ ] 職業理解フィードバックが表示される
- [ ] 推奨シナリオが直接404導線になっていない

## Final Human Questions

- [ ] 学生が次に何をすればよいか迷わないか
- [ ] 大学職員に見せられる完成度か
- [ ] 企業人事にto B拡張のイメージを説明できるか
