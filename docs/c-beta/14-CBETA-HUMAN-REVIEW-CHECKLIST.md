# Cβ Human Review Checklist

## 1. workMaterial統一チェック

- concreteWorkArtifactが残っていないか
- workMaterialが各sceneにあるか

## 2. 5タスク固定チェック

- 1シナリオが5 scenesか
- 1 sceneが1つの実務判断単位になっているか
- 無理な分割や水増しがないか

## 3. Anti-Generic Stakeholder Coordinationチェック

- ステークホルダー調整そのものが主役になっていないか
- 仕事材料を外しても成立するシーンになっていないか
- 職種名だけ変えても成立しないか
- roleWorkKernelが明確か
- roleWorkKernelが構造化objectで、input/output/transformation/nonGenericReasonを説明できるか
- 各sceneのcoreDecisionPrimitiveが職種固有か
- 各sceneのkernelConnectionがscenario-level roleWorkKernelとつながっているか
- roleSpecificMaterials / roleSpecificMetrics があるか
- その職種固有の失敗リスクがあるか

## 4. 学生体験チェック

- 診断テストではなく仕事体験になっているか
- 結果画面が点数ではなく職業理解になっているか
- 誤解修正があるか
- 他職種との違いが学生に伝わるか
