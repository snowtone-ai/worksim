## Role Differentiation Notes

All selected roles were produced as artifact-led scenarios. Every scene has numeric `workMaterial`, `roleSpecificity.kernelConnection`, and role-specific metrics. Generic stakeholder coordination is treated as pressure, not the main decision primitive.
